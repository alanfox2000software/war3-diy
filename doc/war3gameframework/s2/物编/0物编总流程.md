# 关于物编的实现
1. 配置时 lua注册物编
2. 编译时 生成物编ini
3. 运行时 读取物编
## 物编配置
- `shortcut.lua`物编配置的注册
```lua
-- 读取配置信息
yo.ini = {}
require "config.config_loader"
```
## 物编编译

```lua

    -- 全局快捷方式

    -- 重新生成物编(成品在魔兽根目录，替换工程同名文件重新打包后生效)
```
