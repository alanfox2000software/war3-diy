TexCoord

格式 TexCoord Left, Right, Up, Bottom,

例子 TexCoord 0, 1, 0, 0.125,

作用: Uses only a fraction of the given Image. the given Numbers are factors from 0 to 1. with left 0.0 and right 0.5 Up 0 and bottom 1; The Left Side of an Image would be taken skiping the Right part from the center. That gained Fraction Image is that stretched to the frames Size. It is suggested to open the used image with something like GIMP select the wanted space change to show percent.

作用: One can produce a tiled image using numbers bigger than 1 for right and bottom.
