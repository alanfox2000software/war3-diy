FrameTypes

下面是支持的FrameTypes,通常大写开头写在双引号中.

  • BACKDROP
  • BUTTON
  • CHATDISPLAY
  • CHECKBOX
  • CONTROL
  • DIALOG
  • EDITBOX
  • FRAME
  • GLUEBUTTON
  • GLUECHECKBOX
  • GLUEEDITBOX
  • GLUEPOPUPMENU
  • GLUETEXTBUTTON
  • HIGHLIGHT
  • LISTBOX
  • MENU
  • MODEL
  • POPUPMENU
  • SCROLLBAR
  • SIMPLEBUTTON
  • SIMPLECHECKBOX
  • SIMPLEFRAME
  • SIMPLESTATUSBAR
  • SLASHCHATBOX
  • SLIDER
  • SPRITE
  • TEXT
  • TEXTAREA
  • TEXTBUTTON
  • TIMERTEXT

Frames可以简单分为2类: Frames和SimpleFrames. 都可以使用大部分Frame相关的api,只是不能作为父子节点嵌套使用.

SimpleFrames包含如下几个: SIMPLEBUTTON, SIMPLECHECKBOX, SIMPLEFRAME, SIMPLESTATUSBAR以及作为ChildFrame的String and Texture.
