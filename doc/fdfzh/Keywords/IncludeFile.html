IncludeFile

格式 IncludeFile FilePath,

例子 IncludeFile "UI\FrameDef\Glue\StandardTemplates.fdf",

例子 IncludeFile "UI\FrameDef\UI\EscMenuTemplates.fdf",

作用: Allows to inherit Frames from included files. One can add multiple IncludeFile lines. The files mentioned in the IncludeFile list should be sorted by Alphabet (abc)

应用: Fdf
