BackdropCornerFlags

格式 BackdropCornerFlags Text,

例子 BackdropCornerFlags "UL|UR|BL|BR|T|L|B|R", //Whole Border

例子 BackdropCornerFlags "UL|UR|T|L|R", //No Bottom Border

作用: EdgeFiles displayed, one can skip some and order does not matter. Using this without seting border files will crash the game when this Backdrop is displayed

应用: BACKDROP
