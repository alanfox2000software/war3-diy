ControlFocusHighlight

格式 ControlFocusHighlight FrameName,

例子 ControlFocusHighlight "IconicButtonFocusHighlightTemplate",

作用: Define the HIGHLIGHT-Frame beeing displayed while the parentFrame has focus. A Button should have ControlStyle "HIGHLIGHTONFOCUS" to use this.

应用: BUTTON, GLUEBUTTON, TEXTBUTTON, GLUETEXTBUTTON
