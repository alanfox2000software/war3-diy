ControlMouseOverHighlight

格式 ControlMouseOverHighlight FrameName,

例子 ControlMouseOverHighlight "BorderedButtonMouseOverHighlightTemplate",

作用: Define the HIGHLIGHT-Frame beeing displayed while the parentFrame is hovered with the Mouse. A Button should have ControlStyle "HIGHLIGHTONMOUSEOVER" to use this.

应用: BUTTON, GLUEBUTTON, TEXTBUTTON, GLUETEXTBUTTON
